Our Training Programs

Basic Modules (2 Days Training)

1. Assistive Technologies

Assistive Technologies focus on equipping individuals with disabilities with tools that make technology more accessible. This includes screen readers, speech recognition software, and specialized input devices. Trainees learn how to integrate assistive tech into online work environments, ensuring inclusivity and efficiency.

2. Blue Collar Digital Skills

Digital transformation is not limited to office-based jobs. This module empowers blue-collar professionals by teaching them how to market their skills online. Participants learn how to create digital portfolios, leverage social media for visibility, and find clients using online marketplaces.

3. Content Writing

Writing is one of the most in-demand digital skills. This module covers creative and technical writing, blog writing, and copywriting. Participants learn keyword optimization (SEO), how to structure engaging content, and techniques for writing compelling articles that attract readers.

4. Data Entry

This module is designed for individuals looking to start an online career in data entry. It covers accuracy techniques, data organization, working with spreadsheets, and using automation tools to enhance productivity.

5. Digital Marketing & E-Commerce

Participants learn online advertising, brand positioning, and selling products through platforms like Shopify and WooCommerce. Social media marketing strategies and lead generation techniques are also covered.

Advanced Modules (1 Week Training)

1. Basic App Development

Mobile applications power the modern world. This module introduces app development using frameworks like Flutter and React Native. Participants build simple applications while learning coding best practices.

2. Basic Computer Programming

Programming is the foundation of digital transformation. This module covers programming logic, Python basics, and problem-solving techniques essential for developing web and mobile applications.

3. Basic Graphic Design

Visual content is key to branding and marketing. This module teaches design principles using tools like Adobe Photoshop, Canva, and Illustrator. Participants learn logo design, poster creation, and social media graphics.

4. Data Analysis Using Excel

Excel is a powerful tool for business analytics. This module covers data visualization, pivot tables, and basic statistical analysis. Participants also explore automation techniques using macros.

5. Financial Markets and Trading

Led by Moses Kimani, this module dives into forex trading, market analysis, and risk management. Trainees learn how to use TradingView and MetaTrader 5 for technical and fundamental analysis.

6. Introduction to Artificial Intelligence

AI is transforming industries. This module introduces machine learning concepts, automation tools, and AI applications in digital marketing, finance, and customer service.

7. Introduction to Cybersecurity

Cybersecurity is essential for online safety. Participants learn ethical hacking techniques, threat detection, and how to secure digital assets using encryption and VPNs.

8. Introduction to Web Development

Web development powers the internet. This module teaches HTML, CSS, and JavaScript, enabling trainees to build responsive websites from scratch.
